Documentation

2. Template folder structure



  • bootstrap folder contains minified CSS, JavaScript and fonts.
  • css folder contains stylesheets files
  • fonts folder contains fonts for icons used
  • js folder contains custom scripts and plugins call
  • php folder contains script for sending a message
  • plugins folder contains plugins
  • images folder contains all images used in template; each html page has it own subfolder; the subfolder from assets contains the logo and images used in css and js files

3. Bootstrap



Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

bootstrap.min.css

Contains grid systems, default typography and style for tables, forms, images etc.
Read more about Bootstrap css.

bootstrap.min.js

Contains tab, collapse, tooltip, dropdown, scrollspy, carousel etc.
Read more about Bootstrap javascript.

Fonts folder contains fonts for default icons included in bootstrap.

4. Css



The Css files required can be found in the head .

custom.css

Contains typography used in template and style for some elements such as: headlines, alerts, lists, buttons, accordions, tabs, google maps, social media, process steps, pie charts, progress bars, counters, statistics, price plans, testimonials, isotope, filter, pagination, breadcrumb, timeline, full sections, video player, sliders, widgets.

page-style.css

Contains style for header, footer, menu, mobile menu, search, sticky header, page header, elements used in template pages: about me, services boxes, portfolio, blog, comments, comment form, woocommerce elements, contact form.

4.1. Fonts



The template makes use of the Google Fonts service to implement non-standard fonts. Google Fonts makes it quick and easy for everyone to use web fonts, including professional designers and developers. All of the fonts are Open Source.

The font used in template is Raleway.

HTML
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Raleway:400,500,300">
CSS
font: 14px/25px "Raleway", Arial, sans-serif;

4.2. Theme options



You can change the main colors and page header of the template. For that, the folder alternative-style contains the css files that allow it. These files contain all the css rules related strictly to colors and background colors and images.

To change the color you choose one of the files above.

To change page header style you must add class style-1 , style-2 or style-3 at page-header .
